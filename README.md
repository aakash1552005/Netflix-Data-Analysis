> An exploratory data analysis (EDA) project on Netflix's content library to uncover trends, patterns and insights.

---

## 📌 Overview

This project analyzes Netflix's dataset to understand content distribution, popular genres, country-wise production trends, and release patterns over the years.

---

## ✨ Features

- ✅ Content type distribution (Movies vs TV Shows)
- ✅ Top genres & categories analysis
- ✅ Country-wise content production
- ✅ Year-wise content growth trends
- ✅ Top directors & cast analysis
- ✅ Rating distribution insights

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| Python | Core language |
| Pandas & NumPy | Data cleaning & manipulation |
| Matplotlib & Seaborn | Data visualization |
| Google Colab | Development environment |

---

## 📂 Dataset

- **Source:** Netflix Movies and TV Shows (Kaggle)
- **Records:** 8,807 rows × 12 columns
- **Fields:** Title, Type, Director, Cast, Country, Release Year, Rating, Duration, Genre

---

## 📊 Key Insights

- 🎬 Movies make up ~70% of Netflix content
- 📍 USA produces the most content on Netflix
- 📈 Content additions peaked between 2018-2020
- 🎭 Dramas & Comedies are the most popular genres
- 🔞 TV-MA is the most common content rating
